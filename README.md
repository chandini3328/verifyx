VERIFYX is a web-based prototype that helps users verify the credibility of news during emergency situations.

## Problem
Fake news spreads rapidly during crises, causing panic and confusion.

## Solution
VERIFYX uses rule-based logic to analyze news content and generate an explainable credibility score.

## Features
- Simple frontend UI
- Fake / Real credibility check
- Explainable score and reasons
- Deployed live using GitHub Pages

## Technology Used
- HTML
- CSS
- JavaScript (Rule-based logic)

## Status
Final prototype completed. Ready for deployment and demonstration.
